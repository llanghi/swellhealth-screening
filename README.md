# Swell Health Screening Tools

Static website for Swell Health screening questionnaires.

## Version 1.3

- Added shared questionnaire engine (`questionnaire-engine.js`).
- Converted EPDS to shared engine format.
- Added ACOS-Self questionnaire page (`acos.html`).
- Updated screening home page to link EPDS and ACOS-Self.

## Current pages

- `index.html` — screening tools home page
- `epds.html` — Edinburgh Postnatal Depression Scale
- `acos.html` — ADHD Clinical Outcome Scale, Self Report
- `style.css` — shared styling matching the Swell Health Squarespace palette and fonts
